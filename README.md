# Luftverschmutzung-in-Graz

## Ausführen

1. Virtual Environment erstellen:
```bash
python -m venv venv
venv\Scripts\activate
```

2. Dependencies installieren:
```bash
pip install -r requirements.txt
```

3. Analyse ausführen:
```bash
python Main.py
```

## Struktur

- Main.py: Komplette Analyse (Aufgaben 1-7)
- report.pdf: Dokumentation der Ergebnisse
- plots/: Generierte Grafiken
- feinstaubdataexercise.pickle: Rohdaten
